# <기본 문법 문제풀이>  

![0009문제)3과5의배수](../algorithm_image/구름_3과5의배수.png)

풀이)

```
rl.on("line", function(line) {
	var result = 0; 
		for(var i = 1; i <= line; i++) {
			if(i % 3 == 0 || i % 5 == 0) {
				result += i
			}
		}
	console.log(result);
	

	
	
	rl.close();
}).on("close", function() {
	process.exit();
});
```
